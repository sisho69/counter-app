# my-counter-app

Vue 3で作成したシンプルなカウンターアプリです。
カウントの増減やリセットができ、ローカルストレージに自動保存されるため、ページを閉じても前回のカウントが引き継がれます。

デモサイト : https://sisho69.github.io/counter-app

---

# 概要

HTML,CSS,Vue3のみで構成された軽量なwebアプリです。
フレームワークの基礎理解を目的に作成し、Vueのリアクティブなデータ操作やローカルストレージ連携を学ぶための練習として開発しました。

制作期間は約１か月程度

---

# 主な機能

- カウントの増減
- リセット機能
- ローカルストレージへの自動保存
- レスポンシブデザイン対応

---

# 使用技術

- フロントエンド : HTML,CSS,Vue 3
- ビルドツール : Vite
- その他 : ローカルストレージ
